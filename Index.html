HTML semântico

Uso de header, main, section e footer para melhorar a organização.

Layout responsivo

O design se adapta a diferentes tamanhos de tela, tanto para desktop quanto para celular.

Conteúdo bem apresentado

Textos claros, títulos visíveis e navegação simples mostram aprendizado básico de UX para provas e trabalhos.
